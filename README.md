# ID Gallery One-Page Presentation Site

A mobile-first adaptive long-scroll website (slides 001-026 sequence) with large typography, premium visuals, and subtle scroll animations.

## Run locally

```bash
python3 -m http.server 8080
```

Open <http://localhost:8080>

## Deploy to Railway

1. Push this repository to GitHub.
2. In Railway, create a new project from the repo.
3. Railway will auto-detect the `Dockerfile` and deploy the static site.

No build step is required.
